
機能
- ほしい物を登録できる
- ほしい物を一覧で管理できる
- ほしい物の詳細を確認できる
- ほしい物を買った記録をつけることができる
- ほしいと思ってから買ったまでの日数を知ることができる
- ほしい物があといくらで買えるか確認することができる
- 日々我慢すべきことのリストを作成することができる
- 我慢した分の貯金をすることができる
- 貯金した金額を確認することができる
- 貯金した金額から欲しいものを買うことができる
- 貯金額と購入額のグラフを表示することができる(Summaryの表示)
- いつ、どれだけ貯金したかを確認することができる(Summaryの表示)

画面
- ウェルカム画面
- Top画面
- 欲しい物一覧
- 欲しい物登録
- 欲しい物参照
- 欲しい物編集
- 我慢一覧
- 我慢登録
- 我慢参照 
- 我慢編集
- Summary(グラフ:貯金額と使用額)
- Summary(日毎の貯金額)


画面ごとの機能
- ウェルカム画面: Fragment
	-> 初期化
		-> 未定
	-> Tap to Start: Button
		-> Top画面へ遷移

- Top画面: Fragment
	-> 欲しい物: Button
		-> 欲しい物一覧へ遷移
	-> 我慢: Button
	-> Summary: Button
	-> 設定
		-> 未定

- 欲しい物一覧: Fragment
	-> 登録されている欲しい物の一覧を表示: ListView
	-> 追加: Button
	-> 編集: Button

- 欲しい物登録: Dialog
	-> 入力枠
		-> 名称
		-> 金額
	-> 登録: Button
	-> キャンセル: Button


- 欲しい物参照: Fragment
	-> 表示
	-> 編集: Button
		-> 欲しい物編集画面へ遷移
	-> 削除: Button
		-> 削除ダイアログを表示

- 欲しい物編集
	-> 入力フィールド
		-> 値を入力
	-> 更新ボタン
		-> 変更を確定
	-> キャンセルボタン
		-> 何もせずに画面を閉じる
		-> 戻るボタンを同じ

- 削除ダイアログ: Dialog (Common)
	-> OK
		-> 選択したアイテムを削除
	-> Cancel
		-> 何もせずにDialogを閉じる
